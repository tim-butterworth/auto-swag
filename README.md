## Starting the app

# Without swagger

- `./gradlew bootRun`

# With swagger

- `SPRING_PROFILES_ACTIVE=swagger ./gradlew bootRun`

[swagger](http://localhost:8080/swagger-ui.html)

additional documentation can be found [here](http://springfox.github.io/springfox/docs/current/)
